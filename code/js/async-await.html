# async和await用法

  • # async

  1. 函数前面加上 async 关键字，则该函数会返回一个结果为 promise 的对象。

  2. async 函数返回 promise 对象的状态。

  3. 如果返回的是一个非 Promise 类型的数据， 则async 函数返回 promise 的状态 为 fulfilled 成功。

  4. 如果返回的是一个 Promise对象，则 async 函数返回 promise 的状态由返回的Promise对象的状态决定。

  5. 如果 throw Errow 抛出异常，则 async 函数返回 promise 的状态为 rejected 失败。

async function test(){
    //1、如果返回一个非Promise类型的数据，则async函数返回promise的状态为fulfilled成功。
    return 'ok'
    //2、如果返回的是一个Promise对象，则async函数返回Promise的状态为fulfilled。
    return new Promise((resolve, reject) => {
        resolve('ok')
        // reject('error')
    })
    // 3、如果抛出异常，则async函数返回promise的状态为rejected失败。
    throw 'error'
}

let result = test()
console.log(result)

  • # await

1.await 右侧的表达式一般为 promise 对象。

2.await 是在等一个表达式的结果，等一个返回值（回调成功的内容）

3.如果 await 右侧是一个 promise 对象，则 await 返回的是 promise 成功的值。

注意：

1.await 必须写在 async 函数中，但 async 函数中可以没有 await。

2.如果 await 的 promise 失败了，就会抛出异常，该异常需要通过 try catch 捕获处理。

3.如果它等到的是一个 Promise 对象，await 就忙起来了，它会阻塞后面的代码，等着 Promise 对象 resolve，然后得到 resolve 的值，作为 await 表达式的运算结果。

// await只能在async内部，而async可以没有await
async function test(){

    let promise = new Promise((resolve,reject)=>{
        resolve('ok')
    })
    // 通过try catch捕获结果
    try {
        // 接收promise成功后的内容
        let p1 = await promise
        console.log('p1')
    }
    catch(error){
        // 接收错误返回值
        console.log(error)
    }
    
}
